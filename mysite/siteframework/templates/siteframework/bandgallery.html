ONLINE-BAND, 2013

Back To Work Page

Online band is a cross-platform music web application built with Node.js, Express.js, Socket.io and PhoneGap.
The making of Online band is a experience of crazy impossible problem solving. I was responsible for the implementation of the whole backend of the application.

This application is browser-based and bulit into different mobile platforms using PhoneGap. We solved the problem of audio playing latency in browsers by using native code plugins for PhoneGap to pre-load the audios. The backend of the application uses Socket.io to send messages among users to instruct the playing of different notes on all the band members' devices.
