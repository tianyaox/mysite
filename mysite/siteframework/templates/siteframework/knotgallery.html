KNOT, 2013

Back To Work Page

KNOT is a digital namecard social network web application built with Django, HTML5, MySQL and AWS. I was responsible for most of the UI designing/implementing, django template writing, javascript ajax for form submitting tasks. I also participated in the writing of backend models and views.

I designed and implemented the name card of the app. By using css tricks like animation/transforms, the simple-looking card contains complex information and functions for users to interact with. Users can resize or flip each piece of the card, and do functions like adding and editing their information.
