Route Finder, 2014

Back To Work Page

This project is a cooperation with asset managers/civil engineers in CMU. The drawings are loaded as USGSOverlay on Google Maps. Ajax helps with the switch among floors. Shortest path algorithm is used to find the way for users among the pre-entered location points. I participated in the writing of Google Maps API and designing/ implementing of the UI.

This web app is browser based and cross-platform. This is a responsive designed UI, which will transform into the suitable looking when opened on other portable devices.

I also designed and implemented the responsive introduction webpage of the product. By designing this page, I found the beauty of using frosting glass filter effect on title backgrounds.
